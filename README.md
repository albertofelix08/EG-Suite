EG Suite — Engineering Graphics Suite
An interactive 3D learning platform for Engineering Graphics, built for the ME22201 syllabus.
🔗 Live App → eg-suite.vercel.app

What it covers
Unit III & IV — Projection of Solids, Section & Development

Rotating object method for axis inclination
Cutting plane with live section and true shape
Lateral surface development with unroll animation
First-angle orthographic projections (4 views)

Unit V — Isometric Projections & Orthographic Views

Isometric, orthographic, and split 4-view layouts
Angled and frustum cuts with live geometry updates
Construction lines and dimension annotations
PNG export

All rendering is WebGL-powered via Three.js, and the app runs fully offline — no external dependencies at runtime.
